七种内建类型 - 类型名 - typeof返回值(七种)
* `null` - object
* `undefined` - undefined
* `boolean` - boolean
* `number` - number
* `string` - string
* `object` - object (唯一不是基本类型)
* `symbol` - symbol (ES6加入)

* `typeof function a(){} === "function"` funtion类型属于object;
